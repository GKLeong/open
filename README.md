# open
## 1. 概述
### 基于springboot的后台基础开发框架
### 所用技术栈
  - springboot
  - mybatis-plus
  - 阿里云文件存储，消息推送，sms短信发送
  - 海康威视摄像头控制
  - modbus数据获取
  - 网易云信接口接入
  - 微信支付，V2，V3
  - xxxJob基于代码的任务控制
  - zlmedkit视频处理
  - minio
  - mqtt
  - mybatis-plus模板方式生成dao到controller包括测试代码
  - 引入therapi-runtime-javadoc,javadoc生成swagger
  - 引入alibaba easyexcel完成excel生成
  - satoken进行权限相关认证

## 2. 支持平台
  - unix
  - windows
